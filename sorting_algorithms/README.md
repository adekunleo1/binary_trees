# sorting_algorithms
Sorting Algortihms 

This project is done during **Full Stack Software Engineering journey** at **ALX**. Its objective is to learn about sorting algorithms (bubble sort, insertion sort, selection sort, quick sort and so on) and Big O Notation in **C languages.


## Files
All of the following programs are written in C:

| Filename | Description |
| -------- | ----------- |
